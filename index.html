Project #2

The Assessment Assimilator

The Problem

Creating QTI compliant assessments is no easy task. With complex schemas and question types it is cumbersome to build, store, modify and transport assessments.

Solution:

A web application that will create QTI compliant assessments and store them in the cloud.

Once stored you can modify, view and take the assessment.

The ability to import assessments from XML.

The ability to export different types of QTI assessments modules (ePUBs for example).
